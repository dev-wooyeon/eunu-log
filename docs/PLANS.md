# PLANS

Last updated: 2026-02-26

## Planning Model

1. Create active plans in `exec-plans/active/`.
2. Track residual debt in `exec-plans/tech-debt-tracker.md`.
3. Move delivered plans to `exec-plans/completed/` with validation notes.

## Required Fields for Any Plan

1. Clear scope and non-goals.
2. Observable acceptance criteria.
3. Test and rollback strategy.
4. Ownership and ETA assumptions.
