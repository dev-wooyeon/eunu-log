# Core Beliefs

1. Content readability comes first.
2. Interaction should support comprehension, not distract from text.
3. The visual system should be token-driven and consistently reusable.
4. Motion is optional and should degrade safely for reduced-motion users.
5. Mobile ergonomics are first-class, including safe-area-aware spacing.
6. Design decisions must be testable with visual or behavioral checks.
