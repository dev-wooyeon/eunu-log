# UI Component Guide

## Button

### Variants

- `primary`: 핵심 CTA
- `secondary`: 보조 액션
- `tertiary`: 텍스트 버튼

### Sizes

- `sm`, `md`, `lg`

### States

- `loading`
- `disabled`
- `fullWidth`

## EmptyState

### Variants

- `default`: 일반 빈 상태
- `search`: 검색 빈 결과
- `error`: 오류 안내

### Sizes

- `sm`, `md`

### Action

- 클릭 액션: `action.onClick`
- 링크 액션: `action.href`
