# Archive Policy

Use this directory for documentation that is no longer an active source of truth
but should be retained for historical context.

## What Belongs Here

1. Auto-generated analysis snapshots.
2. One-off investigation outputs.
3. Obsolete drafts superseded by active docs.

## What Should Not Be Archived

1. Current operating guides.
2. Product specs in active use.
3. Build/test/security baseline documents.
