# PR Workflow

- PR 작성 시 `/Users/noah/workspace/personal/eunu.log/.github/pull_request_template.md` 템플릿을 그대로 사용합니다.
- `변경 내용 / 의도 / 영향 범위 / 검증`을 모두 채웁니다.

## 필수 체크

- [ ] `npm run build`
- [ ] (필요 시) `npm test`
- [ ] 다크/라이트 모드 확인
- [ ] 모바일/데스크톱 확인

- 브랜치: `codex/<task>` 형식 사용
- 커밋: `<type>: <한국어 설명>` 형식 사용
- 관련 이슈/PR 링크를 본문에 명시
